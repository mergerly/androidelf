**重要的节信息如下：**
.text 代码段 存储编译后的机器指令
.data 数据段 存储初始化的全局变量和局部变量
.bss 数据段 存储未初始化的全局变量和局部变量
.rodata 只读数据段
.rel.text 重定位的代码段，重定位的地方在.text段内，以offset指定具体要定位位置。
.rel.dyn 重定位全局变量，重定位的地方在.got段内。主要是针对外部数据变量符号。
.rel.plt 重定位的地方在.got.plt段内（注意也是.got内,具体区分而已）。主要是针对外部函数符号。
.comment 注释信息段
.plt 保存函数的地址
.plt.got 
.got 保存全局变量引用的地址
.got.plt 用来保存函数引用的地址
.dynamic 动态链接信息
.debug 调试信息

## gdb调试

### 调试准备

##### 1、关闭SELinux
adb shell su
setenforce 0
getenforce
返回Permissive或Disable

##### 2、安装gdbserver x86
cd /d "D:\SDK\Android\SDK\ndk-bundle\prebuilt\android-x86\gdbserver"
adb push gdbserver /data/local/tmp/
adb shell chmod 777 /data/local/tmp/gdbserver

adb pull /system/bin/app_process D:\SDK\symbols\Android\gdb

adb pull /system/lib/libstdc++.so D:\SDK\symbols\Android\gdb
adb pull /system/lib/libm.so D:\SDK\symbols\Android\gdb
